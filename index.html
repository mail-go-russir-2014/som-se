    Welcome to Application of Self-Organizing Maps for Search Engine Practice Session

    Installation Instructions

    If you are working under Ubuntu or other Linux distribution you may skip until the Install SOMToolbox section.

    If you are working under Mac OS X

    On Mac OS X you need to install gnu coreutils. One of the way to do it is using homebrew If you do not have it installed run the following command from the terminal (âŒ˜ + t) 

    $ ruby -e "$(curl -fsSL https://raw.github.com/Homebrew/homebrew/go/install)"

    To install coreutils invoke

    $ brew install coreutils

    Install SOMToolbox

    You may follow installation instructions at the official The Java SOMToolbox webpage. Or download the latest release from here. To do so from terminal run:

    $ mkdir somtoolbox
$ cd somtoolbox
$ wget http://www.ifs.tuwien.ac.at/dm/download/somtoolbox_0.7.5-3.svn4280.tar.gz
$ tar -xzf somtoolbox_0.7.5-3.svn4280.tar.gz

    Copy tutorial data files

    You may follow the links at the top of the page to clone the current repository. Or (assuming you are still in sometoolox directory) run:

    $ git clone https://github.com/mail-go-russir-2014/som-se.git data
